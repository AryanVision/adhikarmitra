## How to Deploy

### Step 1: Enable GitHub Pages
1. Go to: https://github.com/manikantahari99/AdhikarMitra/settings/pages
2. Under "Source", select: **Deploy from a branch**
3. Under "Branch", select: **develop** and folder: **/ (root)**
4. Click **Save**
5. Wait 2-5 minutes for deployment

### Step 2: Verify Deployment
- Visit: https://manikantahari99.github.io/AdhikarMitra/
- It will automatically redirect to src/index.html

## Custom Domain (Optional)
If you want a custom domain like `adhikarmitra.in`:
1. Buy domain from GoDaddy, Namecheap, etc.
2. Add CNAME file with your domain
3. Configure DNS settings
4. Enable in GitHub Pages settings
